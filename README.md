📊 Super Store Sales Dashboard (Power BI)
🚀 Project Overview

This project presents an interactive Sales Dashboard built using Power BI to analyze business performance across different dimensions such as profit, 
sales, category, segment, and payment mode.
📌 Key Features
💰 Displays total Profit (175.26K) and Sales (1.57M)
📈 Monthly trend analysis of Sales and Profit (2019–2020)
🛒 Sales breakdown by Category and Sub-Category
👥 Customer segmentation insights (Consumer, Corporate, Home Office)
💳 Payment mode analysis (Cards, COD, Online)
🚚 Shipping mode performance comparison
📊 Insights Generated

Identifies top-performing categories and products
Tracks monthly growth trends in sales and profit
Highlights customer purchasing behavior
Helps in understanding preferred payment and shipping methods

🛠️ Tools & Technologies
Power BI (Data Visualization)
Data Cleaning & Transformation
DAX (Data Analysis Expressions)
